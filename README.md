# Dark-mod-form
Tela de formulário com modo dark

Projeto de uma página de inscrição para newsletter, desenvolvido com **HTML, CSS e JavaScript puro**.

A proposta do projeto é oferecer uma interface simples e agradável para que usuários possam se cadastrar e receber novidades por e-mail.

## Funcionalidades

- formulário de inscrição com campos de nome e e-mail
- layout limpo e centralizado
- botão para alternar tema
- estrutura simples e fácil de entender
- projeto responsivo para diferentes tamanhos de tela

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
